ПОЯСНИТЕЛЬНАЯ ЗАПИСКА
к проекту «Bugilla!»

1. Введение
Цель проекта
Сделать платформер для “убивания времени” на время долгой загрузки.

Основные задачи
1. Отдых во время перерыва или долгой загрузки
2. Создание геймплея классического платформера


2. Технологии
Используемые технологии
Язык программирования Python

Библиотека Arcade для графики и игровой логики

Формат PNG для спрайтов персонажей и объектов

Использование баз данных для хранения данных уровней и сохранений


3. Игровой процесс
Основные механики
Игрок управляет персонажем, который перемещается по платформам, собирает рубли и уничтожает врагов. Цель каждого уровня — собрать все рубли на локации и достичь финишного флага.

Управление
Управление осуществляется с помощью стрелок для перемещения, пробела для прыжка и клавиши Escape для вызова меню.

Игровые объекты

Player — главный персонаж с возможностью бегать и прыгать

PMC — патрулируют платформы, уничтожают игрок при контакте

Platform — поверхности для перемещения

Rubles — коллекционные предметы, увеличивающие счет

Flag — конечная точка уровня


4. Техническая реализация
Физика игры
Реализована система гравитации, влияющая на падение персонажа. Прыжки работают по классической схеме платформеров: при нажатии пробела персонаж получает вертикальный импульс, после чего постепенно замедляется под действием гравитации.

Система коллизий
Используется встроенная система проверки столкновений между спрайтами. При контакте с платформой игрок останавливает падение, при столкновении с врагом получает урон или уничтожает врага, если приземляется на него сверху.


5. Пользовательский интерфейс
В верхней части экрана отображается информация о количестве собранных рублей и текущем уровне. В игре присутствуют экраны меню, завершения уровня и окончания игры.


6. Заключение
Реализованные функции
В проекте успешно реализованы все основные компоненты классического платформера: физика движения, система врагов, сбор предметов, несколько уровней сложности и пользовательский интерфейс.


Системные требования
Операционная система: Windows

Python версии 3.8 или выше
